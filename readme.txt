== Description ==

Sidebar Widget for your blog. The top headlines about the Florida Marlins and frequent blog posts from Marlins writer Juan Rodriguez. 

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `sun-sentinel-marlins.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Drag Marlins News and Blog Widget to your Sidebar

== Frequently Asked Questions ==
- How do I upload sun-sentinel-marlins.php to the /wp-content/plugins/ directory?
- Use your FTP manager, browse to the /wp-content/plugins/ directory of your blog and upload sun-sentinel-marlins.php to that folder.
